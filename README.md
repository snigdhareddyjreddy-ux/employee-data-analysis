# Employee Data Analysis

## Project Description
This project analyzes employee data using Python and the Pandas library.

## Features
- Load employee data from a CSV file
- Calculate average salary
- Count employees by department
- Filter employees based on salary threshold
- Export filtered employees to a new CSV file

## Technologies Used
- Python
- Pandas
- CSV

## Files
- employee_analysis.py
- employee_data.csv
- filtered_employees.csv (generated after running the program)
